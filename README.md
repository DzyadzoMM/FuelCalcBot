# 🧮 Telegram Конвертер Валідностей (MPG & Miles Converter)

Асинхронний Telegram-бот на базі **aiogram 3.x**, створений для швидкого переведення американських автомобільних величин (MPG, милі) у звичні для європейського ринку метричні значення (літри на 100 км, кілометри).

## 🚀 Основні можливості

* **Конвертер MPG у Л/100 км**: Автоматичний розрахунок за формулою:  
  $$L/100km = \frac{235.215}{MPG}$$
* **Конвертер Миль у Кілометри**: Швидке переведення відстані чи швидкості ($1 \text{ миля} = 1.60934 \text{ км}$).
* **FSM (Конечні автомати)**: Покрокове очікування та обробка введення даних від користувача.
* **Строга валідація вводу**: Захист від введення букв, спецсимволів або некоректних чисел через регулярні вирази (RegEx). Автоматична заміна коми на крапку для зручності.
* **Сучасний UX**: Інтуїтивне керування за допомогою Inline-кнопок.
* **Готовність до деплою**: Адаптований для розгортання на хмарах (Render, Railway тощо).

---

## 🛠 Технологічний стек

* **Мова розробки:** Python 3.10+
* **Фреймворк:** [aiogram 3.x](https://github.com/aiogram/aiogram) (Асинхронна бібліотека для Telegram Bot API)
* **Менеджер станів:** FSMContext (вбудований в aiogram)
* **Валідація:** Регулярні вирази (`re`)
